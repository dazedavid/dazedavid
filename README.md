## 🤖 Highlight: Automated Vulnerability Remediation

<div align="center">


Built a **fully autonomous security remediation pipeline** using **GitHub Copilot Coding Agent** + custom SKILL.md files:

- 🎯 **Zero human intervention** — from scheduled trigger to merged PR
- 🌐 **Multi-ecosystem** — Java (Gradle 9.x), Node.js (npm), Python (uv/pip)
- 🧠 **Custom Skills** — SKILL.md teaches Copilot repo-specific conventions
- 🔒 **Safeguards** — gated version bumps, author filters, branch protection
- ⚡ **4 workflows** — security-remediation → setup → issue-assign → auto-merge

<details>
<summary><b>🔍 How it works (click to expand)</b></summary>

1. **`security-remediation.yml`** — Weekly cron trigger (Monday 06:00 UTC)
2. **`copilot-setup-steps.yml`** — Installs Node.js 20, Python 3.12, uv, configures JFrog registries
3. **`issue-assign-copilot.yml`** — Assigns Copilot agent via GraphQL with Claude Sonnet 4.6
4. **Copilot reads `SKILL.md`** — Runs `npm audit`, `uv audit`, `./gradlew dependencyUpdates`
5. **Copilot fixes code** — Updates deps, removes integrity hashes, bumps version (if changed)
6. **`copilot-auto-merge.yml`** — Merges PR after CI passes (with draft→ready + clean status fallback)

</details>
